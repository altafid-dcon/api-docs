Quick Example

Get Started in Minutes

Authenticate and create your first task with just a few API calls.

  1. Login to get your access token
  2. Use the token in your API requests
  3. Create tasks, manage contacts, and more
View Full Guide ‚Üí 
# 1. Login
curl -X POST 'https://api.altafid.dev.altafid.net/api/auth/login' \
  -H 'Content-Type: application/json' \
  -d '{
    "username": "your-email@example.com",
    "password": "your-password"
  }'

# 2. Create a task
curl -X POST 'https://api.altafid.dev.altafid.net/api/tasks' \
  -H 'Authorization: Bearer YOUR_TOKEN' \
  -H 'Content-Type: application/json' \
  -d '{
    "taskName": "Follow up with client",
    "taskType": "EMAIL",
    "priority": "HIGH",
    "status": "OPEN"
  }'
